## 📌 About

**Task Manager** is a full-stack web application built with **Django 6** that helps individuals stay productive by organizing their daily tasks in one clean, distraction-free interface. Create tasks, track their status, and recover deleted ones — all without the noise.

> *"Manage tasks without stress. Never miss a deadline."*

---

## ✨ Features

### ✅ Current Features

| Feature | Description |
|---|---|
| 🔐 **User Authentication** | Register, log in, and log out securely |
| 📝 **Task Management** | Create, edit, and organize tasks with descriptions and due dates |
| 🗑️ **Recycle Bin** | Soft-delete tasks and restore them anytime — no accidental data loss |
| 📊 **Status Tracking** | Mark tasks complete or incomplete and monitor daily progress |
| 🧹 **Clean Interface** | Distraction-free UI built for focus and productivity |

### 🚀 Upcoming Features

| Feature | Description |
|---|---|
| 🔗 **Task Sharing** | Share and assign tasks to friends or teammates |
| 👥 **Team Collaboration** | Create teams, manage projects, and track group progress |
| 💬 **Comments & Chat** | Discuss tasks with real-time comments and team chat |

## 🚀 Getting Started

### Prerequisites

- Python 3.10+
- pip

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/Dhanashree-Raut/Task-Manager-Django.git
cd Task-Manager-Django
```

**2. Create and activate a virtual environment**

```bash
python -m venv venv

# On Windows
venv\Scripts\activate

# On macOS/Linux
source venv/bin/activate
```

**3. Install dependencies**

```bash
pip install -r requirements.txt
```

**4. Apply database migrations**

```bash
python manage.py migrate
```

**5. Run the development server**

```bash
python manage.py runserver
```

**6. Open in your browser**

```
http://127.0.0.1:8000/
```
